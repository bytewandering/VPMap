This repository contains the implementation of **Towards Precise Guidance: A Novel Cross-Dimensional Mapping Framework for 3D Cerebrovascular Surgical Navigation**.

**Overview**

VPMap is a 2D/3D mapping framework for cerebrovascular surgical navigation. It aims to align intraoperative 2D DSA images with preoperative 3D vascular models by combining vascular anatomical information and projective priors.

The framework mainly includes:

2D vessel segmentation from DSA images
3D vascular model processing
2D/3D vascular registration
Projection-based mapping for surgical navigation
